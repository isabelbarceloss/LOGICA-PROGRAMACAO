# Projeto - SoftGlow
Projeto desenvolvido para a aula de Lógica e Programação.

### `Desenvolvedores`
> Isabel Barcelos, Isabela Pereira, João Diniz e Raquel Layane.

## `Sobre o Projeto:`
O projeto **SoftGlow** tem como objetivo apresentar, por meio de um site visual e intuitivo, uma marca fictícia de skincare desenvolvida especialmente para fins acadêmicos.

O site foi criado utilizando HTML, CSS, JavaScript, PHP e Banco de Dados, organizando as informações de maneira clara e estética para representar a identidade da marca.

Ele apresenta desde a história e propósito da SoftGlow até seus produtos, dicas de cuidados com a pele e interação por meio de formulário.

## `O que o Projeto Aborda:`
- [x] Conceito e identidade da SoftGlow, falando sobre o propósito, missão e valores da marca;
- [x] Apresentação dos produtos de skincare;
- [x] Conteúdos educativos sobre cuidados com a pele;
- [x] Estruturação completa de páginas com HTML e CSS;
- [x] Uso de imagens ilustrativas geradas pelo Gemini;
- [x] Interações com formulários utilizando PHP e Banco de Dados (validar, editar, excluir);
- [x] Navegação simples e organizada para melhor experiência do usuário;
- [x] Estilos modernos voltados ao tema de beleza e autocuidado.
---
**O projeto permitiu explorar tanto conceitos de desenvolvimento web quanto elementos de design voltados à criação de uma identidade de marca.**

**Além disso, reforçou a importância da organização, da experiência do usuário e da construção visual dentro de sites. A SoftGlow representa um exemplo prático de como aplicar técnicas de HTML, CSS, JS PHP e Banco de Dados em um projeto criativo, funcional e bem estruturado.**
