# Projeto - Segurança da Informação
Projeto desenvolvido para a aula de Lógica e Programação.

### `Desenvolvedores:`
> Isabel Barcelos, Isabela Pereira e João Diniz.

## `Sobre o Projeto:`
O projeto **Segurança da Informação** tem como objetivo apresentar, de forma didática e interativa, os principais conceitos relacionados à proteção de dados e boas práticas no ambiente digital. 

O site foi desenvolvido com HTML, CSS e JavaScript, dividido em páginas que abordam desde os princípios básicos até as ameaças mais comuns e medidas preventivas. 

Além de informar, o projeto busca conscientizar os usuários sobre a importância de proteger informações pessoais e profissionais, incentivando o uso de senhas fortes, autenticação em duas etapas e atenção a golpes cibernéticos.

## `O que o Projeto Aborda:`
- [x] Conceito de Segurança da Informação;
- [x] Importância e objetivos, como proteção de dados e prevenção de fraudes;
- [x] Os 3 pilares da segurança: *Confidencialidade*, *Integridade* e *Disponibilidade*;
- [x] Principais ameaças digitais;
- [x] Boas práticas para proteção online;
- [x] Casos reais de ataques cibernéticos;
- [x] Dicas interativas com JavaScript;
- [x] Portfólio dos desenvolvedores.
---
**O projeto proporcionou uma melhor compreensão sobre os conceitos e a importância da segurança da informação.**

**Além disso, o trabalho reforçou a importância de proteger dados pessoais e corporativos, destacando que a segurança digital é uma responsabilidade compartilhada entre usuários e empresas.**
