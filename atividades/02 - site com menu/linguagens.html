Python

  • Criada em 1991 por Guido van Rossum.
  • Conhecida por sua simplicidade e código limpo.
  • Usada em inteligência artificial, ciência de dados, análise de dados, automação e até desenvolvimento web.

JavaScript

  • Criada em 1995, inicialmente para navegadores.
  • Hoje é usada tanto no front-end (parte visual dos sites) quanto no back-end (servidores) com Node.js.
  • É essencial para páginas interativas e dinâmicas.

Java

  • Criada em 1995 pela Sun Microsystems (atualmente Oracle).
  • Muito usada em aplicativos Android, sistemas corporativos e bancos.
  • Seu lema é “escreva uma vez, rode em qualquer lugar”.

C#

  • Criada pela Microsoft em 2000.
  • Muito usada para jogos na Unity, além de softwares para Windows e aplicações empresariais.
  • É considerada moderna e robusta.

C++

  • Criada em 1979 por Bjarne Stroustrup.
  • É uma extensão da linguagem C, mas mais poderosa.
  • Muito usada em jogos, softwares de alto desempenho, sistemas operacionais e robótica.
