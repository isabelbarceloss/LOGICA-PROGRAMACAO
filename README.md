# Aula de Lógica e Programação - UMC
Esse repositório reúne todas as atividades feitas durante as aulas. O objetivo é mostrar o aprendizado básico de programação web com HTML, CSS, PHP, JavaScript e Banco de Dados.

## `O que foi desenvolvido:`
- [x] Páginas web estilizadas com CSS;
- [x] Estruturas básicas de lógica;
- [x] Formulários em HTML/CSS para input de dados;
- [x] Processamento de formulários com PHP;
- [x] Inserção e recuperação de dados com Banco de Dados.
